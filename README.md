# Sales & Profit Dashboard

An interactive Tableau dashboard analyzing sales and profit performance across a national retail dataset of 10,000+ orders.

## Overview

This project explores order-level retail data to uncover sales and profit trends across product categories, time periods, and geography. It was built to give business stakeholders a self-serve, drill-down view of performance without needing to run manual reports.

## Key Metrics

- **10,194** orders analyzed
- **₹19.3 Cr+** in total sales
- **₹2.4 Cr+** in total profit
- **9** states covered
- **3** product categories

## Dashboard Components

- **Bar Chart** — category and sub-category performance comparison, highlighting top and bottom performers
- **Dual-Axis Line Chart** — quarterly sales vs. profit trend over time
- **Filled Map** — state-wise geographic distribution of sales and profit
- **Calculated Field** — a custom `Cost` metric derived as `Sales − Profit`
- **Cross-Filter Actions** — clicking any category, quarter, or state filters all other views, enabling interactive drill-down from a high-level summary to individual states

## Tools Used

- **Tableau Desktop** — dashboard design and calculated fields
- **Tableau Hyper Extract** — underlying data storage for performance
